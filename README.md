# Лабораторная работа №1–2 — Знакомство с OpenGL

Этот репозиторий содержит реализацию лабораторной работы по дисциплине **«Компьютерная графика»**, посвящённой знакомству с базовыми функциями OpenGL и работе с трёхмерной графикой.  
Проект реализован на **C++** с использованием **OpenGL**, **GLU**, **GLAUX** и **WinAPI**.

---

## Цель работы
Ознакомиться с синтаксисом основных функций OpenGL и построить 3D-объект заданной формы с наложением текстуры.

---

## Задание
Необходимо разработать программу, которая:
- строит геометрическую фигуру по заданному образцу;
- разбивает её поверхность на треугольники и четырёхугольники;
- накладывает текстуру на каждую грань;
- отображает и вращает объект в 3D-пространстве;
- выполняет рендеринг с использованием OpenGL fixed pipeline.
